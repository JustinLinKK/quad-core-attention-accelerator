# syn_sram_176b_w16

Release contents for this block:

- `verilog/sram_176b_w16.v`: RTL snapshot used for synthesis
- `run_dc.tcl`: Design Compiler run script
- `sram_176b_w16.sdc`: top-level constraint file kept at the run root for script compatibility
- `constraints/sram_176b_w16.sdc`: duplicated constraint file in a release-friendly location
- `sram_176b_w16.out.v`: mapped output netlist

This folder currently does not include saved synthesis reports under `log/`.
